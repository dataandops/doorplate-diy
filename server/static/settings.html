Flash this sign

The sign's firmware is a generated ESPHome config scoped to this room. First flash is over USB; after that, the sign picks up updates over WiFi via OTA — no cable needed.

  1. Download the config for this room:
    Download doorplate-<id>.yaml
  2. Move it next to your secrets file so the !secret includes resolve: 
    mv ~/Downloads/doorplate-<id>.yaml esphome/
  3. Flash via USB (plug the driver board into the machine running this command first): 
    esphome run esphome/doorplate-<id>.yaml
    Or the helper: make flash ROOM=<id>.
After the first USB flash, esphome run uses OTA automatically — no cable required for updates.

ESPHome dashboard

If you run the ESPHome dashboard, you can edit this sign's config there and push OTA updates from a web UI.
